# Mockups Directory

This directory is reserved for design mockup files.

## Purpose
Place your UI/UX design mockups here to reference during development:
- Color palette references
- Layout designs
- Visual style guides
- Screen mockups

## Usage
When adding new design elements:
1. Place mockup files in this directory
2. Reference them when updating [constants.js](../../constants.js) for colors
3. Use them as a guide when modifying [styles.css](../../styles.css)
